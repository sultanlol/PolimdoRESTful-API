Panduan Menggunakan RESTful API Mock

Di halaman ini, kami akan menjelaskan bagaimana menggunakan API mock ini dengan beberapa contoh menggunakan fetch

1. Menggunakan Fetch untuk Mengakses API

Anda dapat menggunakan fetch di JavaScript untuk melakukan berbagai operasi HTTP seperti GET, POST, PUT,dan DELETE. Berikut adalah contoh penggunaan fetch untuk masing-masing metode:

1.1. Mengambil Data (GET)

Untuk mengambil data dari endpoint, Anda bisa menggunakan metode GET seperti berikut:

fetch('https://jsonplaceholder.typicode.com/posts')
    .then(response => response.json())
    .then(json => console.log(json))
    .catch(error => console.error('Error:', error));

Contoh ini akan mengambil semua posting dari endpoint /posts dan menampilkan hasilnya di konsol.

1.2. Menambahkan Data (POST)

Untuk menambahkan data baru ke server, gunakan metode POST seperti di bawah ini:

fetch('https://jsonplaceholder.typicode.com/posts', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify({
        title: 'Post Baru',
        body: 'Ini adalah konten dari post baru.',
        userId: 1,
    }),
})
    .then(response => response.json())
    .then(json => console.log(json))
    .catch(error => console.error('Error:', error));

Contoh ini mengirimkan data baru ke endpoint /posts dan menampilkan responsnya di konsol.

1.3. Memperbarui Data (PUT)

Untuk memperbarui data secara keseluruhan, gunakan metode PUT:

fetch('https://jsonplaceholder.typicode.com/posts/1', {
    method: 'PUT',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify({
        id: 1,
        title: 'Judul Post Diperbarui',
        body: 'Ini adalah konten yang diperbarui.',
        userId: 1,
    }),
})
    .then(response => response.json())
    .then(json => console.log(json))
    .catch(error => console.error('Error:', error));

Contoh ini memperbarui data pada endpoint /posts/1 dengan data baru yang dikirimkan.

1.4. Menghapus Data (DELETE)

Untuk menghapus data dari server, gunakan metode DELETE:

fetch('https://jsonplaceholder.typicode.com/posts/1', {
    method: 'DELETE',
})
    .then(() => console.log('Data berhasil dihapus'))
    .catch(error => console.error('Error:', error));

Contoh ini menghapus data dari endpoint /posts/1.

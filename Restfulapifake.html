Jalankan kode program ini untuk menguji API:

fetch('http://localhost:3000/api/apimataKuliahTE/2')
    .then(response => response.json())
    .then(json => console.log(json));

Berhasil! Kamu menjalankan skriptnya.

Endpoints API Tersedia

  • /api/apiAkhirPenelitian - List of Akhir Penelitian
  • /api/apiAkhirPengabdian - List of Akhir Pengabdian
  • /api/apiPenelitian - List of Penelitian
  • /api/apiPengabdian - List of Pengabdian
  • /api/apiDosen - List of Dosen
  • /api/apimataKuliahTE - List of Mata Kuliah Teknik Elektro
